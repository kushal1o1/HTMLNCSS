CASCADING STYLE SHEETS

There are mainly three level of style sheets

  1. Inline
    • is specified for a special occurence of a tag and apply only to that tag
    • appears it self in a tag
    • only applicable to one tag only where it is used
    • used in tag like ,h1="color:red";margin:sth px "
      • As a example we can use in this heading

      THIS HEADING IS CREATED USING INLINE CSS USING COLOR AND MARGINLEFT

  2. Document-level / internal
    • apply to whole document in which they appear
    • appear in head of the document
    • It is used by tag style in the head and inside in designing is done

    3. This is done using style tag as internal css just by making class and calling that class inside style tag by simply tag.class

  3. External style
    • Two or more website can use the pre made style by simply calling it
    • but have to notice that the the attached file must not contain any html tags and file must have css extension
    • This is done like this using link tag with rel="stylesheet" href="file.css"
